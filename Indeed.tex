\section{Scraping Indeed Listings}\label{scraping-indeed-listings}}

The goal of this project is to scrape information off of a job listing
on Indeed's website. This process would include the user inputting a
city of their choice and returning a CSV file containing information of
data science job listings in the first 10 pages of the search result.
This will make the job searching process much easier for the user and
the data scraped from the can also be used to create an awesome
dataframe for some data science project. An example would be to find and
compare the salaries and ratings quartiles of data scientist positions
in different cities.

Through this project, I hope to gain exposure to web scraping through
the use of BeautifulSoup.
